## Jak přidat zpracovaný zdroj

1. Zpracujte surový zdroj pomocí `agents/research-synthesizer.md`.
2. Uložte výstup do této složky.
3. Pojmenujte ho: `[stránka-portálu]-[téma].md`
   - Příklady: `grant-zprava-zdroj-fail.md`, `administrativa-docx-zpracovano.md`
4. Zapište ho do tabulky níže.
5. Po integraci do stránky portálu označte stav.
