# Sources / Raw — Surové zdrojové materiály

Tato složka obsahuje surové, neupravené zdroje — přesně tak, jak přišly z výzkumu nebo stažení. Nic se zde neupravuje.

---

## Jak přidat nový zdroj

1. Uložte soubor do této složky.
2. Pojmenujte ho popisně: `[téma]-[datum]-[zdroj].md` nebo `[téma]-[datum]-[zdroj].pdf`
   - Příklady: `grant-reporting-2026-04-notebooklm.md`, `copilot-navod-ms-docs.pdf`
3. Zapište ho do tabulky níže.
4. Předejte `agents/research-synthesizer.md` ke zpracování.
